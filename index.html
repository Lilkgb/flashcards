Intro to Programming Vocabulary:

JavaScript (JS)

is a programming language that manipulates the DOM (Document Object Model) and makes your webpage interactive.

Operators

a special character or characters that indicates an action to be performed.

Ex: +, -, *, /, and % (modulo)

Variables

can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.

Variable naming conventions

assigning a name to your variable. Variable names are case-sensitive and should always start with a letter. If your name is more than one word, use Lower Camel Case.

Ex: myName

Functions

perform actions in the browser.

Methods

an action run on a piece of data. Think of it as a message you send to a piece of data, and the result is the response.

Ex: .toExponential(); .toFixed(); toPrecision();

Arguments

Some methods take one or more arguments that provide the method with additional information to help it perform its action.

Parameters

The parameters, in a function call, are the function's arguments. The parameters hold the argument.

Return

The return value is the method's response.

Ex: undefined, string, boolean.

Chaining methods

Calling a method directly on the return value of another method.

Ex: "foo".concat("bar").toUpperCase();

Strings

a line of text. Using the function toString(): converts a number to a string.

Booleans

Returns true or false. These are not strings.

Undefined

an object (such as a variable) without a defined value.

NaN

Stands for Not A Number. However, it is a type of number.

Escape

These marks \\. It lets you use "" or () in a string and tells the computer that the string isn't over yet and this just adding grammatical punctuations.

Alert

A function. it opens a dialog box and returns undefined.

Comments

Anything following // on a line in JavaScript is ignored by the computer. Comments are useful for leaving notes in your code for yourself and other programmers.

jQuery

Is a library that JavaScript uses/calls on to make programming interactivity easier.

Attributes

A method used to return the attribute with the specified name of an element.
